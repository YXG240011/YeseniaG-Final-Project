## Description
This will give you a series of questions which you will answer. Based on the responses you give in the quiz-like questionnaire, you will get an artwork based on your responses.

## Features
- Feature 1
	- Questionnaire will ask for user feedback. This utilizes code we learned in the first few weeks.
- Feature 2
	- Creation of artwork which is given, This utilizes code we learned in more recent weeks. 


## Challenges
- Making art in vs code.
- Coding request for user feedback.


## Outcomes
Ideal Outcome:
- The user recieves an artwork they will enjoy based on the responses given to questions in quiz.

Minimal Viable Outcome:
- Make a quiz with user feedback with at least more than one available custom artwork to recieve.
